# C/ C++ 핵심 정리  

## 메모리 크기  
```cpp
int = 4
char = 1
short = 2
long = 4
float = 4
double = 8
long double = 12
```


## 자료형(Data Type)  
### 기본 자료형  
C와 C++ 비슷하다고 보면 됨  
```cpp
int a = 10;        // 정수형  
char ch = 'A';     // 문자형  
double pi = 3.14;  // 실수형  
bool flag = true;  // 불리언형  
```


## 상수와 변수  
### 변수 선언  
*'int' 'char' 'float'* 등의 자료형을 사용해 선언 가능함  
```cpp
int a = 10;
char ch = 'A';
double pi = 3.14;        // 소수점 이하 15자리
float Pi = 3.14;         // 소수점 이하 6자리
```
